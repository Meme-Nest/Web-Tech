# Typecasting
Typecasting is a process of converting one datatype to another datatype.


# Number()
## Number to Number

```js
var n =10
console.log(a)
console.log(typeof(a))
var b = Number(a)
console.log(b)
console.log(typeof(b))
```

## String to Number

```js

console.log(a)
console.log(typeof(a))

```

## Boolean to Number
